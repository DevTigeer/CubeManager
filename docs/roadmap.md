# 개발 로드맵

> **주의**: 이 문서는 초기 Phase 기획 문서입니다.
> 실제 개발은 **[development-guide.md](development-guide.md)** 의 Step-by-Step 순서를 따릅니다.
> development-guide에는 레이어별 개발 순서, 의존성 그래프, 코드 꼬임 방지 규칙이 포함되어 있습니다.

## Phase 개요

| Phase | 내용 | 주요 산출물 |
|-------|------|------------|
| **Phase 0** | 프로젝트 셋업 | 프로젝트 구조, DB, 기본 UI 프레임 |
| **Phase 1** | 핵심 기능 (직원/스케줄) | 직원 관리, 스케줄 타임테이블 |
| **Phase 2** | 매출/예약 연동 | 웹 스크래핑, 매출 관리 |
| **Phase 3** | 급여/근태 | 급여 자동 계산, 출퇴근 |
| **Phase 4** | 부가 기능 | 업무자료, 인수인계, 물품관리 |
| **Phase 5** | 마무리 | 테스트, 배포, 최적화 |

---

## Phase 0: 프로젝트 셋업

### 작업 목록

- [ ] WPF + .NET 8 프로젝트 생성
- [ ] MVVM 구조 셋업 (CommunityToolkit.Mvvm)
- [ ] SQLite 데이터베이스 초기화
- [ ] 전체 테이블 스키마 마이그레이션
- [ ] 메인 윈도우 + 탭 네비게이션 구현
- [ ] 공통 UI 컴포넌트 (다이얼로그, 토스트)
- [ ] 설정 탭 기본 화면
- [ ] 관리자 비밀번호 초기 설정 기능

**의존성**: 없음

## 병렬 진행 가능 구간

```
Phase 0 ──────────┐
                   ├── Phase 1 (스케줄) ──── Phase 3 (급여/출퇴근)
                   ├── Phase 2 (예약/매출) ──┘
                   └── Phase 4 (부가기능) ───── Phase 5 (마무리)
```

- Phase 1과 Phase 2는 병렬 진행 가능
- Phase 4는 Phase 0 이후 독립적으로 진행 가능
- Phase 3은 Phase 1 완료 후 진행 (스케줄 의존)
