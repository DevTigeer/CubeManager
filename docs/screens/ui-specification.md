## 3. 공통 UI 컴포넌트

### 3.1 관리자 인증 다이얼로그

```
┌──────────────────────────────┐
│  🔒 관리자 인증              │
│                              │
│  비밀번호: [••••••]          │
│                              │
│       [취소]  [확인]         │
└──────────────────────────────┘
```

- 스케줄 편집, 급여 수정, 글 삭제 등에서 사용
- 인증 후 5분간 유효 (자동 잠금)

### 3.2 확인 다이얼로그

```
┌──────────────────────────────┐
│  ⚠ 확인                     │
│                              │
│  정말 삭제하시겠습니까?      │
│                              │
│       [취소]  [확인]         │
└──────────────────────────────┘
```

### 3.3 토스트 알림

- 하단 우측에 3초간 표시 후 사라짐
- 성공(초록), 경고(노란), 오류(빨간)

|------|----------|--------|

| 테두리 | `#E0E0E0` | 테이블/카드 테두리 |

| 보조 텍스트 | `#757575` | 부가 정보 |
