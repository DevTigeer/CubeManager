# CubeManager 색상 정책 (Color Policy)

> 이 문서는 프로젝트의 색상 사용 기준을 정의합니다. 모든 UI 코드는 이 정책을 준수해야 합니다.

---

## 색상 체계

### 주색 (Main)
| 역할 | HEX | 용도 |
|------|-----|------|
| **주색** | `#2D3047` | 메인 패널 배경, 기본 영역 |
| **어두운 유사색** | `#1E2335` | 사이드바, 가장 어두운 영역 |
| **밝은 유사색** | `#3F425D` | 카드, 섹션, 입력 필드 배경 |

### 보색 (Complementary)
| 역할 | HEX | 용도 |
|------|-----|------|
| **보색 (주황)** | `#F18A3D` | CTA 버튼, 강조, 선택 행 좌측 바 |

### 삼분색 (Triadic)
| 역할 | HEX | 용도 |
|------|-----|------|
| **청록** | `#47A8D7` | 활성 상태, 링크, 선택 행 배경 |
| **자홍** | `#D747A8` | 특수 강조 (극히 제한적) |

### 중성색 (Neutral)
| 역할 | HEX | 용도 |
|------|-----|------|
| **밝은 회색** | `#F0F0F0` | 표 데이터 행 배경 |
| **교차행** | `#E8E8EC` | 표 교차 행 배경 |

---

## 영역별 배경 계층

```
#1E2335  ██  Background (사이드바, 가장 어두운)
#2D3047  ██  Surface (메인 패널 = 주색)
#3F425D  ██  Card (카드, 섹션, 헤더바)
#4A4D68  ██  Border (구분선)
#F0F0F0  ██  TableBg (표 배경 — 강한 대비)
```

## 텍스트 색상

| 계층 | HEX | 용도 | 크기 |
|------|-----|------|------|
| **메인** | `#F0F0F0` | 제목, 주요 텍스트 | 16f, 13f, 11f |
| **보조** | `#A0A3B8` | 부제목, 라벨 | 10f |
| **힌트** | `#6E7190` | 캡션, 비활성 | 8.5f, 9f |
| **표 텍스트** | `#1E2335` | 표 내부 데이터 (밝은 배경 위) | 10f |
| **표 헤더** | `#F0F0F0` | 표 헤더 (어두운 배경 위) | 10.5f |

> **규칙**: 모든 텍스트는 Bold. 기존 Bold/Regular 구분은 크기로 대체.

## 사용 규칙

### 절대 금지
- `Color.FromArgb(r, g, b)` 하드코딩 → 반드시 `ColorPalette.*` 사용
- `Color.White`, `Color.Black` 직접 사용 → `ColorPalette.Text`, `ColorPalette.TextWhite` 사용
- `new Font("폰트명", 크기)` → `DesignTokens.Font*` 사용

### 색상 면적 규칙
- **면적 클수록 색이 어둡게** (배경 = 주색/유사색)
- **면적 작을수록 색이 밝게** (포인트 = 보색/삼분색)
- **표 데이터 영역만 밝은 배경** (배경과 대비)

### 표 색상 규칙
- 헤더: `HeaderBg (#3F425D)` + 밝은 텍스트 `TableHeaderText (#F0F0F0)`
- 데이터: `TableBg (#F0F0F0)` + 어두운 텍스트 `TableText (#1E2335)`
- 교차행: `RowAlt (#E8E8EC)`
- 선택행: `SelectedBg (#47A8D7)` + 흰 텍스트
- 선택 바: `Accent (#F18A3D)` 좌측 3px

### 폰트 규칙
| 용도 | 폰트 | 크기 | 스타일 |
|------|------|------|--------|
| 페이지 제목 | Aptos | 16f | Bold |
| 섹션 제목 | Aptos | 13f | Bold |
| 탭/메뉴/버튼 | Aptos | 10.5f | Bold |
| 본문/데이터 | 맑은 고딕 | 10f | Bold |
| 본문 큰 | 맑은 고딕 | 11f | Bold |
| 본문 작은 | 맑은 고딕 | 9f | Bold |
| 캡션 | 맑은 고딕 | 8.5f | Bold |
| 통계값 | Segoe UI | 24f | Bold |

> Aptos 미설치 시 Segoe UI로 자동 폴백.
