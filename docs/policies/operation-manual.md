# 매장 운영 매뉴얼 (Cube Escape 인천구월점)

> 최종 수정: 2026-03-26
> 이 문서는 CubeManager 체크리스트 시스템에 자동 반영됩니다.

---

## 1. 오픈 업무

### 1-1. 환기 (매일 오픈)
- 장기밀매 창문 윗 고리 열기 (**외부 철창문은 열지 않기**)
- 계단 창문 / 출입문 / 큐브 출입문 열어서 **30분 환기**
- 토요일 오픈: 스케줄 단톡에 올리기

### 1-2. 장치 확인 (화, 목 오픈)
- **타워링**: 모든 버튼 작동 확인, 캐비넷 파손 확인 후 수리
- **타이타닉**: 냄비, 버튼, 2층 올라가는 EM락 확인
- **신데렐라**: 땔감 확인
- **장기밀매**: 컵 확인 후 메모
- 모든 테마 **낙서 확인 후 지우기**
- 확인 후 **보고** (인수인계 또는 단톡)

### 1-3. 알코올 물건 닦기 (월, 수, 금, 일 오픈)
- 상자, 자물쇠 열어서 내부 물건 닦기
- 물건 고장 / 자물쇠 고장 / 힌트 종이 교체 필요 시 교체
- 체크리스트에 이상 유무 기록

---

## 2. 운영 중 업무

### 2-1. 손님 힌트 가이드 (필수)
| 시점 | 행동 |
|------|------|
| **15분** | 첫 문제 못 풀면 → 전화로 힌트 유무 물어보고 드리기 |
| **40분** | 다음방 못 갔으면 → 전화로 힌트 유무 물어보고 드리기 |
| **너무 못 풀 때** | 15분→25분→35분→45분→55분 간격으로 힌트, 두 번째 방 넘기고 설명 |

> **워딩**: "다음에 오셔서 같은 테마 이어서 하실 거 아니면 설명 도와드릴까요?"

- **주말 바쁠 때**: 전화로 최대한 넘겨서 들어가서 오래 설명 안 하도록 유의
- 진행률 무조건 넘겨서 **끝까지 설명하기**

### 2-2. 예약 관리
- 예약 시간 **10분 전**까지 미도착 손님 → **전화 연락** 필수

### 2-3. 근무시간 엄수
- 출퇴근 시간 정확히 준수
- 지각 월 3회 이상 시 경고 (HR 알림 자동 발생)

---

## 3. 마감 업무

### 3-1. 테마 비품 체크 및 정비 (월~금 17:30~18:00)
- 모든 테마 열어서 물건 닦기
- 비품 체크: 랜턴 건전지(흰색) 교체
- 장치 한 번씩 작동시켜 작동 유무 확인

### 3-2. 마감 청소 (매일 마감)
- **손에 잘 안 닿는 곳까지** 청소 후 사진 전송
- 월/화/수: **2개 테마**씩, 목: **1개 테마**

#### 테마별 청소 포인트
| 테마 | 청소 포인트 |
|------|------------|
| 신데렐라 | 책상 아래 + 첫방 / 막방 + 서랍장 아래, 호박마차 뒤·위 |
| 타워링 | 첫방 / 막방 (마스크 끼고 러그 털기) |
| 장기밀매 | 첫방 작은 캐비넷 빼서 아래, 마네킹 아래 / 막방 냉장고 밀어서 아래 |
| 집착 | 첫방 / 막방 침대 아래, 냉장고 밀어서 아래 |
| 타이타닉 | 첫방 침대 아래 / 두 번째 방 / 막방 계단 |
| 카운터 | 모니터, 노트북 청소, 물걸레 |
| 로비 | 책상·의자 위 청소, 정수기 근처 자물쇠 두는 곳 아래, 사진기 근처 |

### 3-3. 잔돈 교체 (목요일 마감)
- **천원**: 50장 이상 확보
- **만원**: 10장 이상 확보
- 부족 시 다음 날 오픈 직원에게 인수인계

### 3-4. 금요일 마감
- 주간 정리 및 다음 주 준비

## 5. 체크리스트 시스템 연동

이 문서의 업무 항목은 CubeManager의 **체크리스트 탭**에 자동으로 노출됩니다.
- 요일별 해당 업무만 체크리스트에 표시
- 파트(오픈/마감)에 따라 담당 업무가 다름
- 미완료 시 **HR 알림** 자동 발생 (관리자 설정 기준)
