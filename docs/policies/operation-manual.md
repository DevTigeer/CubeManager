> 최종 수정: 2026-03-27

## 1. 오픈 업무

### 1-1. 환기 (매일 오픈)
- 장기밀매 창문 윗 고리 열기 (외부철창문은 열지않기)
- 계단창문/문/큐브출입문도 열어서 30분환기
- 토 오픈 : 스케줄 단톡에 올리기

### 1-2. 장치 확인 후 보고 (화, 목 오픈)
- 타워링 모든 버튼, 타워링캐비넷 파손된것 확인후 고치기
- 타이타닉 냄비, 버튼, 2층올라가는 이엠락
- 신데 땔감
- 장기밀매 컵 확인후 메모
- 모든 테마 낙서 확인 후 지우기

### 1-3. 알코올로 물건닦기 (월, 수, 금, 일 오픈)
- 상자나, 자물쇠 열어서 내부 물건 닦기
- 이때 물건고장, 자물쇠고장, 힌트종이 교체 필요시 교체

## 3. 마감 업무

### 3-1. 모든 테마 비품 체크 및 정비 (월~금 5:30 - 6:00)
- 모든 테마 : 열어서 물건닦기, 및 비품체크 랜턴건전지 (흰색)교체
- 장치 한번씩 작동시켜서 작동유무확인

### 3-2. 마감 청소 (매일 마감)
- 손에 잘안닿는 곳까지 청소후 사진 전송
- 월화수 2개씩 목 1개

#### 청소 요일 배정
| 요일 | 청소 테마 |
|:----:|----------|
| **월** | 신데, 타타 |
| **화** | 장기, 로비 |
| **수** | 카운터 |
| **목** | 타워링, 로비 |

#### 테마별 청소 포인트
| 테마 | 청소 포인트 |
|------|------------|
| 신데렐라 | 책상아래 + 첫방 / 막방+ 서랍장아래, 호박마차뒤,위 |
| 타워링 | 첫방 / 막방 (마스크끼고 러그털기) |
| 장기밀매 | 첫방 작은캐비넷 빼서 아래, 마네킹 아래 / 막방 냉장고 밀어서 아래 |
| 집착 | 첫방 / 막방 침대아래 ,냉장고 밀어서 냉장고아래 |
| 타이타닉 | 첫방 침대아래 / 두번째방 / 막방 계단 |
| 카운터 | 모니터, 노트북, 청소, 물걸레 |
| 로비 | 책상위 의자위 청소, 정수기 근처 자물쇠두는곳 아래 ,사진기근처 |

## 5. 체크리스트 시스템 연동

이 문서의 업무 항목은 CubeManager의 **체크리스트 탭**에 자동으로 노출됩니다.
- 요일별 해당 업무만 체크리스트에 표시
- 파트(오픈/마감)에 따라 담당 업무가 다름
- 미완료 시 **HR 알림** 자동 발생 (관리자 설정 기준)
