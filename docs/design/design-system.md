# CubeManager 디자인 시스템

> 레퍼런스: 모던 대시보드 UI (카드 기반 + 사이드 네비게이션 + 통계 카드)
> 대상: WinForms + AntdUI (.NET 8)
> 최종 갱신: 2026-03-19

---

## 1. 디자인 원칙

```
① 정보 밀도 — 한 화면에서 핵심 정보를 바로 파악
② 시각적 위계 — 크기·색상·간격으로 중요도 구분
③ 일관성 — 모든 탭에서 같은 패턴 반복
④ 저사양 친화 — 그림자/그라데이션 최소화, GDI+ 더블 버퍼링
```

## 4. 컴포넌트 디자인

### 4.1 통계 카드 (Summary Card)

```
┌─────────────────────────┐
│  📊 오늘 매출             │  ← 라벨 (12px, Text-Secondary)
│                          │
│  ₩ 1,250,000            │  ← 메인 값 (24px Bold, Text-Primary)
│                          │
│  ▲ 12% vs 어제           │  ← 변동 (11px, Success/Danger)
└─────────────────────────┘

• 크기: 가로 25% (4열), 세로 100px
• 배경: #FFFFFF
• 모서리: 8px radius (AntdUI Panel)
• 그림자: 없음 (저사양), 대신 border 1px #E8ECF1
• 패딩: 16px
• 아이콘: 좌상단 24x24, 배경 원형 (색상별 Light 색)
```

#### 탭별 통계카드 내용

```
  오늘 근무자 수  |  이번 주 총 시간  |  내일 근무자  |  빈 시간대

[출퇴근 탭]
```

## 5. 간격 시스템 (Spacing)

```
단위: 4px 기반

xs   =  4px   인라인 요소 간격
sm   =  8px   같은 그룹 내 요소
md   = 12px   관련 섹션 간격
lg   = 16px   카드 내부 패딩
xl   = 20px   카드 간 간격
2xl  = 24px   섹션 간 간격
3xl  = 32px   페이지 여백
```

### 9.3 DataGridView 테마 적용

```csharp
// 모든 DataGridView에 공통 적용
public static void ApplyTheme(DataGridView grid)
{
    grid.BorderStyle = BorderStyle.None;
    grid.BackgroundColor = Color.White;
    grid.GridColor = Color.FromArgb(240, 240, 240);  // #F0F0F0
    grid.RowHeadersVisible = false;
    grid.EnableHeadersVisualStyles = false;
    grid.CellBorderStyle = DataGridViewCellBorderStyle.SingleHorizontal;

    grid.ColumnHeadersDefaultCellStyle = new DataGridViewCellStyle
    {
        BackColor = Color.FromArgb(248, 250, 252),  // #F8FAFC
        ForeColor = Color.FromArgb(107, 114, 128),  // #6B7280
        Font = new Font("맑은 고딕", 10f, FontStyle.Bold),
        Padding = new Padding(8, 0, 8, 0)
    };

    grid.DefaultCellStyle = new DataGridViewCellStyle
    {
        Font = new Font("맑은 고딕", 10f),
        ForeColor = Color.FromArgb(26, 26, 46),  // #1A1A2E
        Padding = new Padding(8, 4, 8, 4),
        SelectionBackColor = Color.FromArgb(227, 242, 253),  // #E3F2FD
        SelectionForeColor = Color.FromArgb(26, 26, 46)
    };

    grid.AlternatingRowsDefaultCellStyle = new DataGridViewCellStyle
    {
        BackColor = Color.FromArgb(250, 251, 252)  // #FAFBFC
    };
}
```
