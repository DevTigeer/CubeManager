# Feature 07: 물품 관리

## 1. 개요

업장에서 사용하는 물품/비품의 재고를 관리하는 기능.
보유해야 하는 수량, 현재 수량, 부족 수량을 표로 관리한다.

---

## 3. 기능 상세

### 3.1 물품 추가

```
┌──────────────────────────────┐
│  물품 추가                   │
├──────────────────────────────┤
│  물품명:   [____________]    │
│  카테고리: [사무용품 ▾]      │
│  보유기준: [____] 개         │
│  현재수량: [____] 개         │
│  비고:     [____________]    │
│                              │
│         [취소]  [추가]       │
└──────────────────────────────┘
```

### 3.2 수량 수정

- 현재수량 셀을 직접 클릭하여 숫자 입력/수정
- 스피너(▲▼) 버튼으로 1개씩 증감
- 수정 즉시 부족수량 자동 재계산

### 3.3 부족 수량 표시

```
부족수량 = 보유기준 - 현재수량
```

| 상태 | 표시 |
|------|------|
| 부족 (> 0) | 🔴 빨간색 숫자 + ▼ 아이콘 |
| 충분 (= 0) | 🟢 초록색 ✓ |
| 초과 (< 0) | 🔵 파란색 숫자 + ▲ 아이콘 |

### 3.4 카테고리

- 드롭다운 필터: 전체 / 사무용품 / 음료 / 청소용품 / 장비부품 / 기타
- 카테고리 추가/수정 가능

### 3.5 삭제

- 행 선택 → 삭제 버튼 또는 우클릭 메뉴
- 확인 다이얼로그 후 삭제
