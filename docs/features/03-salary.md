# Feature 03: 급여 관리

## 1. 개요

직원들의 월간 급여를 자동 산정하는 기능.
스케줄 기반으로 주차별 근무시간을 집계하고, 공휴일 수당/식비/택시비를 반영한다.

---

## 2. 급여 계산 공식

### 2.1 기본급

```
기본급 = 총 근무시간 × 시급
```

### 2.2 공휴일 수당

```
공휴일 수당 = 평일 공휴일 근무시간 × 3,000원
```

- **적용 조건**: 주말(토/일)이 아닌 평일의 공휴일에 근무한 경우
- **한국 공휴일 기준**: 공공데이터포털 API 또는 내장 공휴일 데이터 활용
- 주말과 겹치는 공휴일: 추가 수당 없음 (이미 주말 근무)

### 2.3 식비 (Feature 04 연동)

```
식비 = 6시간 이상 근무일 수 × 식비 단가
```

### 2.4 택시비 (Feature 04 연동)

```
택시비 = 23:30 이후 퇴근일 수 × 10,000원
```

### 2.5 총 급여

```
총 급여(gross) = 기본급 + 공휴일 수당 + 식비 + 택시비
세금(3.3%)    = 총 급여 × 0.033
실수령액(net) = 총 급여 - 세금
```

### 계산 상세 (홍길동 예시)

```
기본급:       165.0h × 10,000원 = 1,650,000원
공휴일 수당:  8.0h × 3,000원   =    24,000원
식비:         6시간↑ 근무 12일  =    60,000원 (12 × 5,000)
```

## 4. 주차 산정 기준

- 한국 월력 기준
- 1주차: 해당 월 1일이 속한 주 (월~일)
- 월 경계: 해당 월에 속한 날짜만 포함
- 5주차: 월에 따라 존재 여부 다름

---

## 5. 공휴일 판정

### 5.1 데이터 소스

- **1순위**: 공공데이터포털 한국천문연구원 공휴일 API
- **2순위**: 내장 공휴일 데이터 (오프라인 시)

### 5.2 한국 법정 공휴일 목록 (내장)

| 공휴일 | 날짜 | 비고 |
|--------|------|------|
| 신정 | 1월 1일 | 고정 |
| 설날 | 음력 1월 1일 ±1일 | 변동 |
| 삼일절 | 3월 1일 | 고정 |
| 어린이날 | 5월 5일 | 고정 |
| 부처님오신날 | 음력 4월 8일 | 변동 |
| 현충일 | 6월 6일 | 고정 |
| 광복절 | 8월 15일 | 고정 |
| 추석 | 음력 8월 15일 ±1일 | 변동 |
| 개천절 | 10월 3일 | 고정 |
| 한글날 | 10월 9일 | 고정 |
| 크리스마스 | 12월 25일 | 고정 |

### 5.3 추가 수당 판정 로직

```
IF 해당 날짜가 공휴일 AND 해당 날짜가 평일(월~금):
    → 시간당 +3,000원 적용
ELSE:
    → 기본 시급 적용
```

## 6. 편집 모드

- 관리자 비밀번호 인증 후 수기 수정 가능
- 수정 가능 필드: 주차별 시간, 식비, 택시비, 공휴일 수당
- 수정 시 `is_manual_edit = true` 플래그 설정
- 수기 수정된 값은 자동 재계산에서 제외 (수동 우선)
- 수기 수정 셀은 노란색 배경으로 구분
