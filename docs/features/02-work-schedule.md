# Feature 02: 근무 스케줄 관리

## 1. 개요

직원들의 월간/주간 근무 스케줄을 관리하는 기능.
타임테이블은 오전 10시 ~ 새벽 1시, 30분 간격으로 운영된다.

---

## 2. 스케줄 기본 정보

| 항목 | 값 |
|------|-----|
| 기준 단위 | 월간 (한국 월력 기준) |
| 운영 시간 | 10:00 ~ 01:00 (익일) |
| 시간 간격 | 30분 |
| 타임 슬롯 수 | 31개 (10:00~01:00) |

### 타임 슬롯 목록

```
10:00, 10:30, 11:00, 11:30, 12:00, 12:30,
13:00, 13:30, 14:00, 14:30, 15:00, 15:30,
16:00, 16:30, 17:00, 17:30, 18:00, 18:30,
19:00, 19:30, 20:00, 20:30, 21:00, 21:30,
22:00, 22:30, 23:00, 23:30, 00:00, 00:30, 01:00
```

---

## 3. 직원 설정 탭

### 3.1 직원 목록 관리

```
┌──────────────────────────────────────────────┐
│  직원 관리                    [+ 추가] [삭제] │
├────┬──────────┬────────┬──────────┬──────────┤
│ ✓  │ 이름     │ 시급   │ 연락처   │ 상태     │
├────┼──────────┼────────┼──────────┼──────────┤
│ ☑  │ 홍길동   │ 10,000 │ 010-xxxx │ 활성     │
│ ☑  │ 김철수   │  9,860 │ 010-xxxx │ 활성     │
│ ☐  │ 이영희   │  9,860 │ 010-xxxx │ 비활성   │
└────┴──────────┴────────┴──────────┴──────────┘
```

- **체크표시(✓)**: 체크된 직원만 근무 스케줄에 추가 가능
- **추가**: 이름, 시급, 연락처 입력
- **삭제**: 선택한 직원 비활성화 (데이터 보존)

## 5. 직원 스케줄 추가

### 5.1 추가 플로우

1. **[+ 직원 추가]** 버튼 클릭
2. 활성 직원 목록 팝업 표시

```
┌──────────────────────────────────┐
│  직원 선택                       │
├──────────────────────────────────┤
│  ○ 홍길동                        │
│  ○ 김철수                        │
│  ○ 이영희                        │
│                    [다음 →]      │
└──────────────────────────────────┘
```

3. 직원 선택 후 근무 시간 설정

```
┌──────────────────────────────────┐
│  홍길동 스케줄 설정              │
├──────────────────────────────────┤
│  출근: [14:00 ▾]                 │
│  퇴근: [19:30 ▾]                 │
│                                  │
│  요일 선택:                      │
│  ☑ 월  ☑ 화  ☑ 수               │
│  ☑ 목  ☑ 금  ☐ 토  ☐ 일        │
│                                  │
│        [취소]  [적용]            │
└──────────────────────────────────┘
```

4. **[적용]** → 선택한 요일에 해당 시간대로 타임테이블 반영

### 5.2 적용 규칙

- 동일 시간대에 여러 직원 배치 가능
- 기존 스케줄이 있는 경우 덮어쓰기 확인 팝업
- 적용 범위: 해당 월의 남은 주차 전체 (또는 선택 주차만)

## 6. 편집 모드

### 6.1 활성화

1. **[편집 모드 🔒]** 버튼 클릭
2. 관리자 비밀번호 입력 다이얼로그

```
┌──────────────────────────────┐
│  관리자 인증                 │
│                              │
│  비밀번호: [••••••]          │
│                              │
│       [취소]  [확인]         │
└──────────────────────────────┘
```

3. 인증 성공 → 편집 모드 활성화 (🔒 → 🔓)

### 6.2 편집 가능 항목

| 작업 | 방법 |
|------|------|
| 이름 입력 | 셀 클릭 → 직원 이름 직접 타이핑 |
| 이름 수정 | 셀 더블클릭 → 수정 |
| 이름 삭제 | 셀 선택 → Delete 키 또는 우클릭 삭제 |
| 범위 선택 | 드래그로 여러 셀 선택 → 일괄 입력/삭제 |

### 6.3 편집 모드 종료

- **[편집 완료]** 버튼 또는 5분 비활동 시 자동 잠금

---

## 7. 월간 뷰

- 월 단위 달력 형태로 각 날짜별 근무 직원 요약 표시
- 날짜 클릭 시 해당 주차 주간 뷰로 이동

---

## 8. 비즈니스 로직

### ScheduleService

```
- GetWeekSchedule(year, month, weekNum) → 주간 타임테이블
- GetMonthSchedule(year, month) → 월간 스케줄 요약
- AddSchedule(employeeId, startTime, endTime, days[]) → 일괄 추가
- UpdateScheduleCell(date, timeSlot, employeeName) → 단일 셀 수정
- DeleteSchedule(scheduleId) → 삭제
- CalculateWeeklyHours(employeeId, year, month, weekNum) → 주간 근무시간
```
