# Feature 05: 업무자료 관리

## 1. 개요

Markdown(.md) 파일 기반의 업무 문서를 관리하는 기능.
검색, 수정, 삭제, 삽입이 가능한 문서 관리 시스템.

---

## 2. 파일 구조

```
data/
└── documents/
    ├── 매뉴얼/
    │   ├── 방탈출-운영-가이드.md
    │   ├── 예약-처리-절차.md
    │   └── 장비-점검-목록.md
    ├── 정책/
    │   ├── 근무-규정.md
    │   └── 환불-정책.md
    └── 기타/
        └── 비상연락망.md
```

## 4. 기능 상세

### 4.1 문서 조회

- 좌측 폴더 트리에서 파일 선택
- 우측에 Markdown 렌더링 결과 표시
- 지원 문법: 제목, 목록, 볼드/이탤릭, 코드블록, 테이블, 이미지, 링크

### 4.2 문서 검색

- **전문 검색**: 제목 + 내용에서 키워드 검색
- **검색 결과**: 파일명, 매칭 줄 미리보기 표시
- **실시간 필터링**: 타이핑 시 즉시 결과 표시

### 4.3 문서 편집

- **[편집]** 버튼 → Markdown 원문 에디터 표시
- 좌우 분할: 왼쪽 에디터 / 오른쪽 프리뷰
- 저장: `Ctrl+S` 또는 저장 버튼
- 자동 저장: 30초 간격

### 4.4 문서 생성

- **[+ 새 문서]** → 폴더 선택 + 파일명 입력
- 빈 MD 파일 생성 후 편집 모드 진입

### 4.5 문서 삭제

- **[삭제]** → 확인 다이얼로그 → 파일 삭제
- 삭제 시 휴지통으로 이동 (30일 보관)

### 4.6 폴더 관리

- 폴더 생성/이름 변경/삭제
- 드래그 앤 드롭으로 문서 이동

## 5. 비즈니스 로직

### DocumentService

```
- GetFolderTree() → 폴더/파일 트리 구조
- GetDocument(path) → Markdown 내용
- SaveDocument(path, content) → 저장
- CreateDocument(folder, fileName) → 새 파일 생성
- DeleteDocument(path) → 휴지통 이동
- SearchDocuments(keyword) → List<SearchResult>
- CreateFolder(parentPath, folderName) → 폴더 생성
- MoveDocument(from, to) → 파일 이동
```
