# Feature 08: 출/퇴근 관리

## 1. 개요

직원의 실시간 출퇴근을 기록하고, 스케줄 대비 지각/조퇴 여부를 색상으로 표시하는 기능.

## 3. 출근 버튼 동작

### 3.1 플로우

1. 직원 선택 (드롭다운)
2. **[출근]** 버튼 클릭
3. 현재 시각 기록
4. 스케줄 대비 판정

### 3.2 판정 로직

```
예정_출근시간 = 스케줄의 출근 시간 (예: 14:00)
실제_출근시간 = 버튼 클릭 시 현재 시각

IF 실제_출근시간 <= 예정_출근시간:
    → 🔵 파란색 글씨 (정상 출근 / 일찍 출근)
    → status = 'on_time'
ELSE:
    → 🔴 빨간색 글씨 (지각)
    → status = 'late'
```

### 3.3 예시

| 예정 출근 | 실제 출근 | 결과 |
|-----------|-----------|------|
| 14:00 | 13:55 | 🔵 파란색 (정상) |
| 14:00 | 14:00 | 🔵 파란색 (정상) |
| 14:00 | 14:01 | 🔴 빨간색 (지각) |
| 14:00 | 14:15 | 🔴 빨간색 (지각) |

## 4. 퇴근 버튼 동작

### 4.1 플로우

1. 직원 선택 (이미 출근 기록 있는 직원)
2. **[퇴근]** 버튼 클릭
3. 현재 시각 기록
4. 스케줄 대비 판정

### 4.2 판정 로직

```
예정_퇴근시간 = 스케줄의 퇴근 시간 (예: 00:00)
실제_퇴근시간 = 버튼 클릭 시 현재 시각

IF 실제_퇴근시간 >= 예정_퇴근시간:
    → 🔵 파란색 글씨 (정상 퇴근 / 늦게 퇴근)
    → status = 'on_time'
ELSE:
    → 🔴 빨간색 글씨 (조퇴)
    → status = 'early'
```

### 4.3 예시

| 예정 퇴근 | 실제 퇴근 | 결과 |
|-----------|-----------|------|
| 00:00 | 23:58 | 🔴 빨간색 (조퇴) |
| 00:00 | 00:00 | 🔵 파란색 (정상) |
| 00:00 | 00:15 | 🔵 파란색 (정상) |

## 5. 스케줄 타임테이블 반영

출퇴근 기록은 스케줄 타임테이블에도 색상으로 반영된다:

```
┌──────────┬──────────┐
│ 시간     │ 수18     │
├──────────┼──────────┤
│ 13:30    │          │
│ 14:00    │ 🔵홍길동 │  ← 출근 셀: 파란색(정상) 또는 빨간색(지각)
│ 14:30    │  홍길동  │
│ ...      │  홍길동  │
│ 19:00    │  홍길동  │
│ 19:30    │ 🔵홍길동 │  ← 퇴근 셀: 파란색(정상) 또는 빨간색(조퇴)
│ 20:00    │          │
└──────────┴──────────┘
```

### 5.1 색상 규칙 (타임테이블)

| 상태 | 출근 셀 | 퇴근 셀 |
|------|---------|---------|
| 정상 | 🔵 파란색 체크 | 🔵 파란색 체크 |
| 지각 | 🔴 빨간색 글씨 | - |
| 조퇴 | - | 🔴 빨간색 글씨 |
| 미출근 | ⚪ 회색 (미기록) | - |

## 7. 야간 근무 처리

- 퇴근 시간이 자정(00:00)을 넘기는 경우:
  - 예: 출근 18:00, 퇴근 01:00 → 정상적으로 처리
  - 날짜는 출근 기준일로 기록
  - 퇴근 시간 비교 시 익일 시간으로 계산

## 8. 비즈니스 로직

### AttendanceService

```
- ClockIn(employeeId, currentTime) → AttendanceRecord
- ClockOut(employeeId, currentTime) → AttendanceRecord
- GetTodayAttendance() → List<AttendanceRecord>
- GetMonthlyAttendance(employeeId, yearMonth) → List<AttendanceRecord>
- JudgeClockInStatus(scheduledTime, actualTime) → 'on_time' | 'late'
- JudgeClockOutStatus(scheduledTime, actualTime) → 'on_time' | 'early'
- GetScheduledEmployees(date) → List<Employee> (오늘 근무 예정 직원)
```
