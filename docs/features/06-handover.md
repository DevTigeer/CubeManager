## 3. 기능 상세

### 3.1 글 작성

```
┌──────────────────────────────────┐
│  인수인계 작성                   │
├──────────────────────────────────┤
│  작성자: [홍길동 ▾]             │
│                                  │
│  내용:                           │
│  ┌──────────────────────────┐   │
│  │                          │   │
│  │ (여러 줄 입력 가능)       │   │
│  │                          │   │
│  └──────────────────────────┘   │
│                                  │
│         [취소]  [등록]           │
└──────────────────────────────────┘
```

- 작성자: 활성 직원 드롭다운 선택
- 내용: 여러 줄 텍스트 입력

### 3.2 댓글

- 각 인수인계 글 하단에 댓글 입력란
- 작성자 이름 + 내용 입력 후 등록
- 시간순 정렬

### 3.3 대댓글

- 댓글의 **[답글]** 버튼 클릭 → 대댓글 입력
- 들여쓰기로 계층 표시
- 깊이 제한: 2단계 (댓글 → 대댓글)

### 3.4 목록

- 최신순 정렬 (기본)
- 페이지네이션: 10개씩
- 검색: 작성자/내용 키워드 검색

### 3.5 수정/삭제

- 자신이 작성한 글/댓글만 수정/삭제 가능
- 관리자: 모든 글 수정/삭제 가능 (관리자 비밀번호)

## 4. 비즈니스 로직

### HandoverService

```
- GetHandovers(page, pageSize, keyword?) → PagedResult<Handover>
- CreateHandover(authorName, content) → Handover
- UpdateHandover(id, content) → bool
- DeleteHandover(id) → bool
- AddComment(handoverId, authorName, content, parentCommentId?) → Comment
- UpdateComment(commentId, content) → bool
- DeleteComment(commentId) → bool
- GetComments(handoverId) → List<Comment> (트리 구조)
```
