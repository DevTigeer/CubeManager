# Feature 04: 식비 / 택시비 자동 집계

## 1. 개요

직원의 근무 스케줄을 기반으로 식비와 택시비를 자동 산정하는 기능.
급여 탭의 이름 옆에 식비/택시비 컬럼으로 표시된다.

---

## 2. 식비 규정

| 항목 | 기준 |
|------|------|
| 지급 조건 | 하루 6시간 이상 근무 |
| 지급 단가 | 설정 가능 (기본 5,000원) |
| 산정 기준 | 스케줄 상 근무시간 기준 |

### 2.1 산정 로직

```
FOR each 근무일 in 해당 월:
    근무시간 = 퇴근시간 - 출근시간
    IF 근무시간 >= 6시간:
        식비_일수 += 1

식비 합계 = 식비_일수 × 식비_단가
```

### 2.2 예시

| 날짜 | 출근 | 퇴근 | 근무시간 | 식비 |
|------|------|------|----------|------|
| 3/1 | 14:00 | 19:30 | 5.5h | ✗ (6h 미만) |
| 3/2 | 10:00 | 18:00 | 8.0h | ✓ |
| 3/3 | 14:00 | 22:00 | 8.0h | ✓ |

---

## 3. 택시비 규정

| 항목 | 기준 |
|------|------|
| 지급 조건 | 23:30 이후 퇴근 |
| 지급 금액 | 10,000원 (고정) |
| 산정 기준 | 스케줄 상 퇴근시간 기준 |

### 3.1 산정 로직

```
FOR each 근무일 in 해당 월:
    IF 퇴근시간 >= 23:30:
        택시비_일수 += 1

택시비 합계 = 택시비_일수 × 10,000원
```

### 3.2 예시

| 날짜 | 퇴근 시간 | 택시비 |
|------|-----------|--------|
| 3/1 | 19:30 | ✗ |
| 3/2 | 23:00 | ✗ |
| 3/3 | 23:30 | ✓ (10,000원) |
| 3/4 | 00:30 | ✓ (10,000원) |
| 3/5 | 01:00 | ✓ (10,000원) |

### 3.3 경계 시간 처리

- 23:30 **이상** (23:30 포함) → 택시비 지급
- 23:29 이하 → 택시비 미지급

---

## 4. 급여 탭 연동

급여 테이블에서 식비/택시비 컬럼으로 표시:

```
│ 이름   │ ... │ 총시간 │ 식비    │ 택시비  │ 공휴일수당 │ ...
│ 홍길동 │ ... │ 165.0  │ 60,000  │ 20,000  │ 24,000     │ ...
│ 김철수 │ ... │  88.0  │ 32,000  │      0  │      0     │ ...
```

- 클릭 시 상세 내역 팝업 (어느 날짜에 식비/택시비 발생했는지)

---

## 5. 설정

`app_config` 테이블에서 관리:

| 설정 키 | 기본값 | 설명 |
|---------|--------|------|
| `default_meal_allowance` | 5000 | 식비 단가 (원) |
| `taxi_allowance` | 10000 | 택시비 금액 (원) |
| `taxi_cutoff_time` | 23:30 | 택시비 기준 퇴근 시간 |
| `meal_min_hours` | 6 | 식비 최소 근무시간 |
