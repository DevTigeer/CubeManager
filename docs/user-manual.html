CubeManager 사용자 매뉴얼방탈출 업장 HR 자동화 + 업무 자동화 시스템 · v0.2.0

CubeManager는 방탈출 카페의 예약·매출·스케줄·급여·출퇴근·인수인계를 하나의 프로그램으로 관리하는 Windows 데스크톱 앱입니다.

💻 시스템 요구사항: Windows 10/11 · .NET 8 런타임 · 4GB RAM · 100MB 디스크

🖥️화면 구성

앱은 항상 전체화면(Maximized)으로 실행되며 4개 영역으로 구성됩니다.

영역위치설명
사이드바좌측 200px11개 탭 네비게이션 (아이콘 + 이름)
헤더상단 50px앱 로고 · 🧮계산기 · 🔄새로고침 · 현재시각 · 지점명
콘텐츠중앙선택한 탭의 내용 (Dock.Fill 반응형)
상태바하단 28px앱 버전 표시

직원 등록
🛡️ 관리자 → 직원관리 → [+ 직원 추가] → 이름, 시급, 연락처 입력

⚙️ 설정 → cubeescape.co.kr 로그인 정보 입력 → 연결 테스트
체크리스트 템플릿 정의
🛡️ 관리자 → 체크리스트관리 → 역할별(오픈/미들/마감) 업무 항목 추가
스케줄 등록
📋 스케줄 → [+ 스케줄 추가] → 직원, 시간, 요일, 주차 선택

예약 표 (메인 그리드)

각 행은 하나의 예약/결제 건입니다. 카드금액·현금금액·계좌금액·비고 컬럼은 직접 편집 가능합니다.

  • 정산 완료 시 맨 오른쪽 체크박스(✓)를 체크
  • 취소/노쇼 건은 취소선 + 회색 표시
  • 기본 정렬: 임박순 (현재시간에 가까운 예약 먼저)

하단 3분할

영역내용
좌측 (40%)지출 내역 그리드 + [지출 추가] 버튼
중앙 (30%)오늘의 통계: 예약수, 취소, 노쇼, 총인원, 객단가
우측 (30%)결제 요약: 카드 현금 계좌 총매출, 지출, 현금잔액

📋스케줄

주간 근무 시간표를 시각적으로 관리합니다.

타임테이블 (GDI+ 커스텀 렌더링)

  • 세로축: 시간 (10:00~익일 03:00, 30분 단위)
  • 가로축: 월~일 (7일)
  • 직원별 색상 블록 + 좌측 4px 컬러바로 식별
  • 오늘 열: 연한 파란 배경 강조
  • 현재 시간선: 빨간 실선 + 좌측 시각 라벨
  • 정시선 = 실선, 30분선 = 점선 (가독성 차별화)

스케줄 추가 다이얼로그

[+ 스케줄 추가] 버튼 또는 빈 셀 더블클릭으로 열립니다.

  1. 직원 선택 — 드롭다운 (활성 직원만)
  2. 근무 시간 — 출근 → 퇴근 (30분 단위 콤보박스)
  3. 적용 기간 — 연/월 + 주차 토글 (1~5주)
  4. 요일 — 월~일 토글버튼 (주말 빨간색)

기존 블록 더블클릭 시 삭제 확인 다이얼로그가 표시됩니다.

하단 주간 요약

직원별 총 근무시간 표시: ● kimjaja 36.5h

⏰출퇴근

직원의 출근/퇴근을 기록하고 스케줄 대비 지각/조퇴를 자동 판별합니다.

출퇴근 입력

  • 직원 콤보박스에서 선택 → [출근] 또는 [퇴근] 클릭
  • Enter = 출근, Shift+Enter = 퇴근
  • 현재 시각이 실시간 표시됩니다

🎫무료이용권

무료 이용권을 발급·관리합니다. 번호는 A2000부터 자동 증가합니다.

발급 폼

이름, 인원수, 연락처, 사유, 비고를 입력 후 [+ 발급] 클릭

사유 태그

  • 월최고기록 — 파란색
  • 장치보상 — 주황색
  • 기타 — 회색

사용 처리

그리드에서 [사용체크] 클릭 → 오늘 날짜로 사용일 기록

📦물품

매장 물품의 필요수량과 현재수량을 관리합니다.

  • 현재수량 셀을 직접 편집하여 재고 업데이트
  • 부족 시 ▼ 부족수량 표시
  • 충분 시 ✓ 0 표시
  • 카테고리: 사무용품, 음료, 청소용품, 장비부품, 기타

📄업무자료

마크다운 문서를 폴더 구조로 관리합니다. 레이아웃은 1:9 비율입니다.

🔑테마힌트

방탈출 테마별 힌트를 관리하고 JSON으로 내보낼 수 있습니다.

  • 좌측: 테마 목록 → 클릭으로 선택
  • 우측: 선택 테마의 힌트 그리드 (텍스트, 카테고리, 순서, 난이도)
  • 우클릭: 테마 수정/삭제
  • [전체 Export] / [선택 Export]: JSON 파일 생성

하위 탭 구성

📊 대시보드

  • 이번 달 매출/지출/지각건수/현금잔액 통계 카드

  • ◀ / ▶ 로 월 이동, [재계산]으로 전체 급여 산출
